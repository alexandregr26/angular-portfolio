Project Description

For my final project in the University of Toronto's second-year computer engineering course, Computer Organization (ECE243), I coded a first-person shooter (FPS) game - my first-ever dive into game development. Drawing inspiration from classic arcade games, the FPS utilized a basic yet effective ray tracing algorithm to render a 3D environment. The game was developed in the C language and fine-tuned for execution on Altera's DE1-SoC board, where users use switches to navigate the virtual space and push buttons to shoot fireballs at enemies. Even though this was my first and only experience with game development, it sparked an interest in me, making me consider future roles in software engineering with a focus on the exciting world of game development.

The source code for the FPS game is available on my GitHub page but on a private repository to support academic integrity. Please contact me if you would like to know more.
